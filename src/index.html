Demo App

This is a demo application to show the use of a very simple material design framework over which rapid prototyping can be done.

With the use of very minimal class tags, we can get a good looking interface to sandbox and test our web application.

Shown in this demo are

  • card.css
  • button.css
  • form.css
  • table.css
